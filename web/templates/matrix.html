Overall Comparison

Key Findings

Multi-Agent and RAG tied at 72% average accuracy

Both outperform Single Agent (59%) significantly. Multi-Agent excels on Arab Barometer (87%) thanks to the Reviewer catching errors. RAG is strongest on GSS (58%) where codebook context helps identify variable names.

Multi-Agent achieves 97% average weight usage

The Reviewer agent explicitly checks for survey weight application, achieving 100% on AB and WVS. RAG achieves 90% across all datasets. Single Agent averages 77%.

RAG is the fastest advanced architecture

RAG averages 84s per question vs Multi-Agent's 160s. Single is fastest at 101s but least accurate. RAG offers the best accuracy-to-speed ratio.

GSS remains the most challenging dataset

All agents score 50-58% on GSS vs 74-87% on other datasets. The 6000+ columns and coded variable names make it harder, though the search_columns tool improved results from the earlier 0-38% range.
